# Bootcamp API

> Backend API for bootcamp directory website

## Run App

```
# Run in dev mode
nodemon server

```

## Database Seeder

To seed the database with users, bootcamps, courses and reviews with data from the "\_data" folder, run

```
# Destroy all data
node seeder -d

# Import all data
node seeder -i
```
